Биномни коефицијент

Број подскупова са k елемената неког скупа, који има n елемената, зове се биномни коефицијент n над k. Биномни коефицијент n над k означаваћемо са binKoef(n,k); рачуна се по формули binKoef(n,k) = (n/k)*binKoef(n-1,k-1) = n!/(k!*(n-k)!). По договору је binKoef(n,0) = 1 и binKoef(n,k) = 0 ако је k<0. Једнакост binKoef(n,k) = 0 је тачна и у случају када је k>n.
